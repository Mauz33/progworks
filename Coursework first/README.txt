Чтобы запустить программу, нужно запустить /Application v0.1/Client/Client.exe и /Application v0.1/Server/Server.exe
Чтобы запустить проект в Qt, нужно открыть файл проекта, находящийся в сооствествуйщей папке в Project

Все необходимые .dll библиотеки, модули и плагины уже находятся в папке с .exe . 
Поэтому достаточно просто запустить .exe файл и приложение запустится без каких-либо проблем.

Список комманд и их краткое описание:

Команды связанные с соединением:
ping - если есть соединение, то вернет pong
echo "message" - если есть соединение, то вернет message
disconnect - отключает от сервера, моментально всплывает диалоговое окно с повощением, что вы отключились

Команды связанный со строками:
set key value - устанавливает значение по ключу, если все хорошо, то вернет ОК
get key - возврщает значение по ключу, вернет value
exist - скажет существует ли ключ
getnset key value - вернет старое значение по ключу и заменит на новое
append key value - складывает value как строки и возвращает текущий результат сложения. Если ключа не существовало до этого, то создаст новую пару ключ-значение и вернет новое значение
increment key - преобразует строку value в int и увеличит на 1, если строка содержит только число,а если строка содержит буквы, то преобразует строку в int равный 0 и увеличит на 1 и вернет резуьтат
decrement key- аналогично increment, только уменьшит на 1 и вернет результат
